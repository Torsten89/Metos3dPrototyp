About in short:

This website should record my approach of exporting Metos3D to Android. This project is part of a module called "master-project" for master of computer science at university of Kiel, Germany.

About Metos3D:

Metos3D is a collection of simulation software, models and data intended to be used for simulation and optimization of marine ecosystem models.

It can be run using only the programming language C and PETSc (Portable, Extensible Toolkit for Scientific Computation).

Why this page?

First, I though this project is not too hard using the Android Native Developer Kit (NDK).

After many hours of trying and sheer mad googling I have realized, that it is way harder than I thought first, due to many reasons:

  • I am not an expert in C-programming and build processes (one goal for me of this project is to learn such things :) )
  • The NDK is still under heavy development and not well documented. For example when I started the official developer guide to the hello world programm, it stated that every NDK-project must declare an Android.mk file and explained how it has to be build. But in the example project wasn't such a file. At that time the Android.mk file was already build automaticly from the Android gradle-build process. (See here)
  • PETSc is a huge toolkit with many dependencies. The installation scripts are an own big framework by themselves and are not supposed to be used with Android. Android does not support the plane GNU Make tool and the configuration scripts are written in Python, which is not supported by Android either. So exporting PETSc on Android is a very difficult task.

Goal of this page:

So the goal of this page is to sort and document the knowledge I gain during the project. I hope that other students, scientist, or somebody out there will profit from this gathered knowledge in the future.
