Dive into Android NDK

After we have realized that compilation of C and Java is done in complete different worlds, we want to dive smoothly into the Android NDK (native developer kit), which combines these two worlds within an Android application. First we are going to take a glance at the Java Native Interface (JNI), which lets us call C-functions within Java. Second the NDK basics will be explained. After that I tried to dive in with a little example-project in Android Studio - and crashed really frustrating. Finally I tried the same with Eclipse and become happy again :)
